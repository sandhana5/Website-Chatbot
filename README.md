# Website-Chatbot
A smart AI-powered chatbot designed to answer user queries based on a structured dataset. This project demonstrates how Natural Language Processing (NLP) can be integrated into a web application to provide automated responses.
Features
 Interactive chatbot interface
 Responds using CSV-based dataset
 Basic NLP-based query matching
 Fast and lightweight backend (Flask)
 Easy integration into any website
 Scalable for future AI improvements
